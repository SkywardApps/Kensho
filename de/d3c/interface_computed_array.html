Detailed Description

A ComputedArray is an array that proxies another.

This is a basic transformation - so a 1-1 entity relationship, generally speaking, but the entities in this proxy may (and should) be of a different type than the underlying.

Basically, this provides for a way to create a ViewModel from a DataModel
