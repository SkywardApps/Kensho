Detailed Description

The IObservable protocol allows any object to be 'observed' in the Kensho system.

This is a read-only value.

Observables must be a Number, String, List, Map, or generic Object. Lists and Maps are treated as 'collections'. Any observable can be observed by delegates implementing the Observer protocol, and collections can emit additional events to delegates implementing the CollectionObserver protocol.

All observables must return an object from the 'value' property.
