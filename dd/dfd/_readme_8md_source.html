37 ## Introduction
38 Knockout is a JavaScript library that helps you to create rich, responsive display and editor user interfaces with a clean underlying data model. Any time you have sections of UI that update dynamically (e.g., changing depending on the userâ€™s actions or when an external data source changes), KO can help you implement it more simply and maintainably.
39 
40 ### Headline features:
41 - Elegant dependency tracking - automatically updates the right parts of your UI whenever your data model changes.
42  Declarative bindings - a simple and obvious way to connect parts of your UI to your data model. You can construct a complex dynamic UIs easily using arbitrarily nested binding contexts.
43  Trivially extensible - implement custom behaviors as new declarative bindings for easy reuse in just a few lines of code.
44 
45 ### Additional benefits:
46 
47 - Pure JavaScript library - works with any server or client-side technology
48  Can be added on top of your existing web application without requiring major architectural changes
49  Compact - around 13kb after gzipping
50  Works on any mainstream browser (IE 6+, Firefox 2+, Chrome, Safari, others)
51  Comprehensive suite of specifications (developed BDD-style) means its correct functioning can easily be verified on new browsers and platforms
